# Number Analyzer JS 🚀

A beginner-friendly JavaScript project that demonstrates how loops work using practical examples and console output.

## Features

- Count numbers using loops
- Detect even and odd numbers
- Calculate totals using accumulators
- Generate multiplication tables
- Create patterns with nested loops
- Loop through arrays using `for...of`
- Practice `for` and `while` loops

# How to Run the Project

## 1. Install Node.js

Download and install Node.js:

https://nodejs.org

---

## 2. Create the Project Folder

```bash
mkdir number-analyzer-js
cd number-analyzer-js
```

---

## 3. Create the JavaScript File

Create a file named:

```bash
app.js
```

Paste the project code inside it.

---

## 4. Run the Project

Open terminal inside the project folder and run:

```bash
node app.js
```

# What the Program Does

## Number Counter
Prints numbers from 1 to 20.

## Even and Odd Checker
Checks whether numbers are even or odd.

## Sum Calculator
Calculates the sum from 1 to 50.

## Multiplication Table
Displays the multiplication table of 7.

## Countdown Timer
Uses a `while` loop to count down before launch.

## Triangle Pattern
Uses nested loops to print a star pattern.

## Array Loop
Loops through an array of programming languages.
